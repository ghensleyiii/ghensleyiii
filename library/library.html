Code Library

Explore the core concepts of 16 programming languages, organized into clear, concise libraries. Each language section provides collapsible dropdowns with one-sentence explanations of fundamental elements, helping you master the essentials of coding.

HTML

Structural Tags

<html>

The <html> tag is the root element of an HTML document, containing all other elements.

<head>

The <head> tag contains metadata, links to resources, and scripts for the HTML document.

<body>

The <body> tag holds the visible content of the web page, such as text, images, and forms.

<title>

The <title> tag sets the title of the web page, displayed in the browser tab.

Content Tags

<p>

The <p> tag defines a paragraph, organizing text into readable blocks with automatic spacing.

<h1> to <h6>

The <h1> to <h6> tags define headings, with <h1> being the most important and <h6> the least.

<a>

The <a> tag creates a hyperlink, enabling navigation to other pages or resources using the href attribute.

<img>

The <img> tag embeds an image, requiring a src attribute for the file path and alt for accessibility.

<ul>

The <ul> tag creates an unordered list, displaying items with bullets.

<ol>

The <ol> tag creates an ordered list, displaying items in a numbered sequence.

<li>

The <li> tag defines an item within a <ul> or <ol> list.

Semantic Tags

<header>

The <header> tag defines introductory content, typically containing logos or navigation.

<footer>

The <footer> tag defines the bottom section of a page, often containing copyright or contact information.

<article>

The <article> tag represents self-contained content, like a blog post or news story.

<section>

The <section> tag groups related content, providing structure to a document.

<nav>

The <nav> tag defines a set of navigation links, improving accessibility and structure.

Attributes

id

The id attribute assigns a unique identifier to an element for styling or scripting purposes.

class

The class attribute applies shared styles or behaviors to multiple elements.

src

The src attribute specifies the source file for elements like <img> or <script>.

alt

The alt attribute provides a text description for images, used by screen readers and when images fail to load.

href

The href attribute specifies the URL destination for <a> tags or linked resources.

style

The style attribute applies inline CSS styles directly to an element.

Form Elements

<form>

The <form> tag creates a container for user input elements, enabling data submission.

<input>

The <input> tag creates an input field, with types like text, checkbox, or radio for various data.

<button>

The <button> tag creates a clickable button, often used to submit forms.

<label>

The <label> tag describes an <input> element, improving accessibility and usability.

<textarea>

The <textarea> tag allows multi-line text input, ideal for comments or messages.

<select>

The <select> tag creates a dropdown menu for selecting one or multiple options.

Formatting Tags

<strong>

The <strong> tag emphasizes text, typically rendering it bold to indicate importance.

<em>

The <em> tag emphasizes text, typically rendering it italic to indicate stress.

<br>

The <br> tag inserts a line break within text, useful for formatting.

<hr>

The <hr> tag creates a horizontal rule to visually separate content sections.

Table Elements

<table>

The <table> tag defines a table to organize data into rows and columns.

<tr>

The <tr> tag defines a row within a <table>.

<td>

The <td> tag defines a data cell within a <tr> row.

<th>

The <th> tag defines a header cell in a <table>, typically bold and centered.

Multimedia Tags

<audio>

The <audio> tag embeds sound files, supporting formats like MP3 with playback controls.

<video>

The <video> tag embeds video files, supporting formats like MP4 with playback controls.

<iframe>

The <iframe> tag embeds external content, such as websites or videos, within a page.

Meta and Link Tags

<meta>

The <meta> tag provides metadata, such as character encoding or page descriptions, for browsers and search engines.

<link>

The <link> tag connects external resources, like stylesheets or favicons, to the document.

Other Elements

<div>

The <div> tag is a non-semantic block-level container for grouping content for styling or scripting.

<span>

The <span> tag is a non-semantic inline container for styling or scripting small text segments.

<script>

The <script> tag embeds or links to JavaScript code to add interactivity.

<style>

The <style> tag defines internal CSS styles for the document.

<!-- -->

The <!-- --> syntax creates a comment, invisible to users but useful for code notes.

CSS

Selectors

.class

The .class selector targets elements with a specific class attribute.

#id

The #id selector targets a single element with a unique id attribute.

element

The element selector targets all instances of a specific HTML tag, like <p> or <div>.

[attribute]

The [attribute] selector targets elements with a specific attribute or attribute value.

*

The * selector targets all elements in the document.

element,element

The element,element selector groups multiple elements to apply the same styles.

element element

The element element selector targets elements that are descendants of another element.

element>element

The element>element selector targets direct child elements of a specified parent.

Properties

color

The color property sets the text color of an element.

font-size

The font-size property defines the size of the text in an element.

font-family

The font-family property specifies the typeface for an element’s text.

margin

The margin property controls the outer spacing around an element.

padding

The padding property sets the inner spacing within an element’s borders.

border

The border property defines the style, width, and color of an element’s border.

background-color

The background-color property sets the background color of an element.

display

The display property specifies how an element is rendered, such as block, inline, or flex.

position

The position property controls the positioning method of an element, like static or absolute.

width

The width property sets the width of an element’s content area.

height

The height property sets the height of an element’s content area.

text-align

The text-align property specifies the horizontal alignment of text within an element.

line-height

The line-height property sets the height of a line of text, affecting vertical spacing.

opacity

The opacity property sets the transparency level of an element, from 0 to 1.

Pseudo-Classes

:hover

The :hover pseudo-class applies styles when a user’s cursor hovers over an element.

:active

The :active pseudo-class applies styles when an element is being clicked.

:focus

The :focus pseudo-class applies styles when an element gains focus, like an input field.

:first-child

The :first-child pseudo-class targets the first child element of its parent.

:last-child

The :last-child pseudo-class targets the last child element of its parent.

Pseudo-Elements

::before

The ::before pseudo-element inserts content before an element’s content.

::after

The ::after pseudo-element inserts content after an element’s content.

::first-line

The ::first-line pseudo-element styles the first line of text in an element.

::first-letter

The ::first-letter pseudo-element styles the first letter of an element’s text.

Units

px

The px unit represents a fixed pixel size for measurements like font-size or width.

%

The % unit represents a percentage relative to the parent element’s size.

rem

The rem unit is relative to the root element’s font size, ensuring consistent scaling.

em

The em unit is relative to the font size of the current element or its parent.

vw

The vw unit represents 1% of the viewport’s width.

vh

The vh unit represents 1% of the viewport’s height.

At-Rules

@media

The @media rule applies styles based on device characteristics, like screen width.

@import

The @import rule includes an external CSS file into the current stylesheet.

@font-face

The @font-face rule defines a custom font for use in the stylesheet.

Python

Keywords

def

The def keyword defines a function in Python.

if

The if keyword introduces a conditional statement to execute code based on a condition.

elif

The elif keyword specifies an additional condition to check if the previous if condition is false.

else

The else keyword defines a block of code to execute if all preceding conditions are false.

for

The for keyword creates a loop to iterate over a sequence, like a list or range.

while

The while keyword creates a loop that continues as long as a condition is true.

break

The break keyword exits a loop prematurely, stopping further iterations.

continue

The continue keyword skips the current loop iteration and proceeds to the next.

return

The return keyword exits a function and optionally returns a value to the caller.

import

The import keyword includes external modules or libraries in a Python program.

from

The from keyword specifies the module or package to import specific components from.

class

The class keyword defines a blueprint for creating objects with attributes and methods.

try

The try keyword begins a block to attempt code that may raise an exception.

except

The except keyword defines a block to handle specific exceptions raised in a try block.

finally

The finally keyword specifies a block that executes regardless of whether an exception occurs.

Built-in Functions

print()

The print() function outputs text or values to the console.

len()

The len() function returns the length of an object, such as a string or list.

range()

The range() function generates a sequence of numbers for iteration.

type()

The type() function returns the data type of a given object.

int()

The int() function converts a value, like a string or float, to an integer.

str()

The str() function converts a value to a string representation.

float()

The float() function converts a value to a floating-point number.

list()

The list() function creates a list from an iterable or converts an object to a list.

dict()

The dict() function creates a dictionary from key-value pairs or converts an object to a dictionary.

input()

The input() function prompts the user for input and returns it as a string.

Data Types

int

The int data type represents whole numbers, positive or negative, without decimals.

float

The float data type represents numbers with decimal points.

str

The str data type represents a sequence of characters, used for text.

list

The list data type is an ordered, mutable collection of items, defined with square brackets.

tuple

The tuple data type is an ordered, immutable collection of items, defined with parentheses.

dict

The dict data type is an unordered collection of key-value pairs, defined with curly braces.

set

The set data type is an unordered collection of unique items, defined with curly braces or set().

bool

The bool data type represents a boolean value, either True or False.

Operators

+

The + operator performs addition for numbers or concatenation for strings and lists.

-

The - operator performs subtraction between numbers.

*

The * operator performs multiplication for numbers or repetition for strings and lists.

/

The / operator performs division, returning a float result.

//

The // operator performs floor division, returning an integer result.

%

The % operator returns the remainder of division between two numbers.

==

The == operator checks if two values are equal.

!=

The != operator checks if two values are not equal.

and

The and operator evaluates to True if both operands are true.

or

The or operator evaluates to True if at least one operand is true.

not

The not operator inverts the boolean value of an expression.
