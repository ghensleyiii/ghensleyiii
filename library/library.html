HTML

Tags

<html>

The <html> tag is the root element of an HTML document, containing all other elements.

<head>

The <head> tag contains metadata and links to resources like stylesheets for the HTML document.

<body>

The <body> tag holds the visible content of the web page, such as text and images.

<div>

The <div> tag is a block-level container used to group content for styling or scripting.

<p>

The <p> tag defines a paragraph, organizing text into readable blocks.

Attributes

id

The id attribute assigns a unique identifier to an element for styling or scripting.

class

The class attribute applies shared styles or behaviors to multiple elements.

src

The src attribute specifies the source file for elements like images or scripts.

alt

The alt attribute provides a text description for images, enhancing accessibility.

CSS

Properties

color

The color property sets the text color of an element.

font-size

The font-size property defines the size of the text in an element.

margin

The margin property controls the outer spacing around an element.

display

The display property specifies how an element is rendered, such as block or inline.

Selectors

.class

The .class selector targets elements with a specific class attribute.

#id

The #id selector targets a single element with a unique id attribute.

element

The element selector targets all instances of a specific HTML tag.

Python

Keywords

def

The def keyword defines a function in Python.

if

The if keyword introduces a conditional statement to execute code based on a condition.

for

The for keyword creates a loop to iterate over a sequence.

import

The import keyword includes external modules or libraries in a Python program.

Built-in Functions

print()

The print() function outputs text or values to the console.

len()

The len() function returns the length of an object, such as a string or list.

range()

The range() function generates a sequence of numbers for iteration.

Keywords

The const keyword declares a constant variable that cannot be reassigned.

Methods

push()

The push() method adds an element to the end of an array.
