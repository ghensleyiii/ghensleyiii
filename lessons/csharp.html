Lesson 50: Dependency Injection

Dependency Injection

C#’s dependency injection (DI) in ASP.NET Core manages service lifetimes with `IServiceCollection`. It promotes loose coupling and testability. DI is a core pattern in modern C# applications.

using Microsoft.Extensions.DependencyInjection;

interface IService
{
    string GetMessage();
}

class Service : IService
{
    public string GetMessage() => "Hello!";
}

class Program
{
    static void Main()
    {
        var services = new ServiceCollection();
        services.AddSingleton();
        var provider = services.BuildServiceProvider();
        var service = provider.GetService();
        Console.WriteLine(service?.GetMessage());
    }
}

Output: Hello!
