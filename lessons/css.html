Lesson 1: What is CSS?

Understanding CSS

CSS, or Cascading Style Sheets, is a language used to style and layout web pages. It controls the appearance of HTML elements, such as colors, fonts, and spacing. Learning CSS is essential for creating visually appealing websites.

p {
    color: blue;
    font-size: 16px;
}

This is styled text.

Lesson 2: CSS Syntax

CSS Syntax

A CSS rule consists of a selector and a declaration block. The selector targets HTML elements, and the block contains properties and values. This structure allows precise control over element styling.

h1 {
    color: red;
    text-align: center;
}

Centered Red Heading

Lesson 3: Inline CSS

Inline CSS

Inline CSS applies styles directly to HTML elements using the style attribute. It’s useful for quick, one-off styling but less maintainable for large projects. External or internal CSS is preferred for scalability.

External CSS

External CSS is stored in a separate .css file, linked to HTML using the `` tag. It promotes reusability and maintainability across multiple pages. This is the preferred method for large websites.

Lesson 6: Element Selectors

Element Selectors

Element selectors target HTML tags directly, applying styles to all instances of that tag. For example, styling all `

` tags changes every paragraph. They are the simplest form of CSS selectors.

div {
    background-color: lightgray;
    padding: 10px;

Lesson 7: Class Selectors

Class Selectors

Class selectors, prefixed with a dot (.), target elements with a specific class attribute. They allow reusable styling across multiple elements. Classes are versatile for applying consistent designs.

.highlight {
    background-color: yellow;
    font-weight: bold;
}

Lesson 8: ID Selectors

ID Selectors

ID selectors, prefixed with a hash (#), target a single element with a unique id attribute. They are ideal for styling specific elements but should be used sparingly. IDs ensure precise, one-off styling.

#unique {
    color: teal;
    font-size: 20px;
}

Lesson 9: Universal Selector

Universal Selector

The universal selector (*) targets all elements on a page. It’s useful for applying global styles, like resetting margins. Use it cautiously to avoid unintended styling.

* {
    margin: 0;
    padding: 0;
}

Lesson 10: Grouping Selectors

Grouping Selectors

Grouping selectors apply the same styles to multiple elements by separating them with commas. This reduces code repetition and improves maintainability. It’s efficient for shared styling needs.

h1, h2, h3 {
    color: darkblue;
    text-align: center;
}

Lesson 11: Box Model

Box Model

The CSS box model defines an element’s structure, including content, padding, border, and margin. Padding adds space inside the border, while margin adds space outside. Understanding the box model is crucial for layout design.

.box {
    width: 100px;
    padding: 10px;
    border: 2px solid black;
    margin: 15px;
}
Box
Lesson 12: Margin

Margins

Margins create space outside an element’s border, separating it from other elements. They can be set individually (e.g., margin-top) or using shorthand (e.g., margin: 10px). Margins are essential for spacing in layouts.

.box {
    margin: 20px 10px 15px 5px;
}
Box with margins
Lesson 13: Padding

Padding

Padding adds space inside an element, between the content and border. It can be set for each side or using shorthand. Padding enhances the visual spacing of content within elements.

.box {
    padding: 15px;
    border: 1px solid gray;
}
Padded box
Lesson 14: Border

Borders

Borders surround an element’s padding and content, defined by width, style, and color. Shorthand properties like border: 1px solid black simplify styling. Borders add visual structure and emphasis.

.box {
    border: 2px dashed red;
}
Dashed border box
Lesson 15: Box-Sizing

Box-Sizing

The box-sizing property controls how the box model calculates width and height. Setting box-sizing: border-box includes padding and border in the total size. This makes layouts more predictable and easier to manage.

.box {
    box-sizing: border-box;
    width: 100px;
    padding: 10px;
    border: 5px solid blue;
}
Border-box
Lesson 16: Display Property

Display Property

The display property controls an element’s rendering behavior, such as block, inline, or none. Block elements take full width, while inline elements flow with text. Setting display: none hides elements entirely.

.inline {
    display: inline;
}
.block {
    display: block;
}
Inline Block
Lesson 17: Position: Static

Static Positioning

Position: static is the default positioning, placing elements in the normal document flow. It ignores top, right, bottom, and left properties. Static positioning is the baseline for other positioning types.

.box {
    position: static;
    background-color: lightblue;
}
Static box
Lesson 18: Position: Relative

Relative Positioning

Position: relative moves an element relative to its normal position using top, right, bottom, or left. It stays in the document flow, affecting nearby elements. Relative positioning is useful for subtle adjustments.

.box {
    position: relative;
    top: 10px;
    left: 20px;
    background-color: coral;
}
Relative box
Lesson 19: Position: Absolute

Absolute Positioning

Position: absolute removes an element from the document flow, positioning it relative to its nearest positioned ancestor. It uses top, right, bottom, or left for placement. Absolute positioning is ideal for overlays or tooltips.

.box {
    position: absolute;
    top: 20px;
    right: 20px;
    background-color: violet;
}
Absolute box
Lesson 20: Position: Fixed

Fixed Positioning

Position: fixed locks an element relative to the viewport, keeping it in place during scrolling. It’s removed from the document flow, unaffected by other elements. Fixed positioning is great for navigation bars or popups.

.box {
    position: fixed;
    bottom: 10px;
    right: 10px;
    background-color: gold;
}
Fixed box
Lesson 21: Colors

CSS Colors

CSS supports colors via names, hex codes, RGB, or HSL formats. Each format offers flexibility for defining hues and opacity. Colors enhance the visual appeal of web designs.

.box {
    background-color: #ff6347;
    color: rgb(255, 255, 255);
}

Lesson 22: Background Properties

Backgrounds

Background properties like background-color, background-image, and background-size style element backgrounds. Shorthand properties combine multiple settings for efficiency. Backgrounds add depth and personality to designs.

.box {
    background: url('image.jpg') no-repeat center;

Lesson 23: Text Alignment

Text Alignment

The text-align property controls horizontal text alignment, such as left, right, center, or justify. It applies to block-level elements containing text. Proper alignment improves readability and aesthetics.

.text {
    text-align: center;
}

Lesson 24: Font Properties

Font Styling

Font properties like font-family, font-size, and font-weight customize text appearance. Shorthand font properties combine multiple settings for brevity. Fonts significantly impact a website’s tone and readability.

.text {
    font: bold 18px Arial, sans-serif;
}

Bold Arial text

Lesson 25: Text Decoration

Text Decoration

The text-decoration property adds effects like underline, overline, or line-through to text. It’s commonly used for links or to emphasize content. Removing decoration (none) is useful for clean link styles.

a {
    text-decoration: none;

.strike {

Lesson 26: Flexbox Basics

Flexbox

Flexbox is a layout model for aligning and distributing items in a container. Setting display: flex on a parent element enables flexible child arrangements. It’s ideal for responsive, one-dimensional layouts.

.container {
    display: flex;
    justify-content: space-between;
}

Lesson 27: Flexbox Alignment

Flexbox Alignment

Properties like justify-content and align-items control flex item alignment along the main and cross axes. Options include center, space-around, and flex-end for versatile layouts. Alignment makes flexbox powerful for dynamic designs.

.container {
    display: flex;
    justify-content: center;
    align-items: center;
}

Lesson 28: Flexbox Wrapping

Flexbox Wrapping

The flex-wrap property controls whether flex items stay on one line or wrap to multiple lines. Setting flex-wrap: wrap allows items to flow naturally. Wrapping is key for responsive layouts with varying content.

.container {

Lesson 29: CSS Grid Basics

CSS Grid

CSS Grid is a two-dimensional layout system for rows and columns. Setting display: grid on a container defines a grid structure. Grid is ideal for complex, structured layouts.

.grid {
    display: grid;
    grid-template-columns: 1fr 1fr;

Grid Template Areas

Grid-template-areas define named regions for grid items, simplifying layout design. Each item is assigned to an area using grid-area. This approach creates intuitive, maintainable layouts.

.grid {
    display: grid;
    grid-template-areas: "header header" "main sidebar";

.header {

Lesson 31: Transitions

CSS Transitions

Transitions smoothly animate property changes, like color or size, over a duration. Properties like transition-property and transition-duration control the effect. Transitions enhance user interactions with subtle animations.

.box {
    background-color: blue;
    transition: background-color 0.3s ease;
}
.box:hover {
    background-color: red;
}

Lesson 32: Transform

Transform Property

The transform property applies 2D or 3D transformations, like rotate, scale, or translate. It’s often paired with transitions for smooth effects. Transforms add dynamic visual flair to elements.

.box {
    transform: rotate(45deg);
}
Rotated
Lesson 33: Animations

CSS Animations

Animations use @keyframes to define a sequence of styles over time. Properties like animation-name and animation-duration control the effect. Animations create engaging, repeatable motion effects.

@keyframes slide {
    from { transform: translateX(0); }
    to { transform: translateX(100px); }
}
.box {
    animation: slide 2s infinite;
}
Sliding
Lesson 34: Pseudo-Classes

Pseudo-Classes

Pseudo-classes, like :hover or :first-child, style elements based on state or position. They add interactivity without JavaScript. Pseudo-classes are essential for dynamic user experiences.

a:hover {
    color: green;
}

Lesson 35: Pseudo-Elements

Pseudo-Elements

Pseudo-elements, like ::before and ::after, style specific parts of an element or insert content. They’re useful for decorative effects or icons. Pseudo-elements enhance design without extra HTML.

p::before {

Lesson 36: CSS Variables

CSS Custom Properties

CSS variables, defined with --prefix, store reusable values like colors or sizes. They’re scoped to elements and accessed with var(). Variables simplify maintenance and theming.

:root {
    --main-color: #3498db;
}
.box {
    background-color: var(--main-color);
}
Variable-colored box
Lesson 37: Media Queries

Media Queries

Media queries apply styles based on device characteristics, like screen width. They enable responsive designs for desktops, tablets, and phones. Media queries are crucial for modern web development.

@media (max-width: 600px) {
    .box {
        background-color: pink;
    }
}

Responsive Units

Units like vw, vh, rem, and em scale relative to viewport or font sizes. They ensure layouts adapt to different screens. Responsive units are key for flexible designs.

.box {

Lesson 39: Overflow

Overflow Property

The overflow property controls content that exceeds an element’s bounds, with options like hidden, scroll, or auto. It’s useful for managing large content in fixed containers. Overflow ensures clean, controlled layouts.

.box {
    width: 100px;
    height: 50px;
    overflow: auto;

The z-index property controls the stacking order of positioned elements. Higher values place elements above lower ones. Z-index is critical for overlapping elements like modals.

Lesson 41: Opacity

Opacity

The opacity property sets an element’s transparency, from 0 (fully transparent) to 1 (fully opaque). It affects the element and its children. Opacity creates subtle visual effects like overlays.

.box {
    opacity: 0.5;

Lesson 42: Box Shadow

Box Shadow

The box-shadow property adds shadows to elements, defined by offset, blur, spread, and color. It enhances depth and visual hierarchy. Shadows are popular for modern, elevated designs.

.box {
    box-shadow: 2px 2px 5px rgba(0, 0, 0, 0.3);

Lesson 43: Border Radius

Border Radius

The border-radius property rounds an element’s corners, creating smooth edges. It can be uniform or varied for each corner. Rounded corners soften designs and improve aesthetics.

.box {
    border-radius: 10px;

Lesson 44: CSS Filters

CSS Filters

Filters apply visual effects like blur, grayscale, or brightness to elements. They’re useful for images or interactive hover effects. Filters add creative flair without image editing.

.box {
    filter: blur(2px);

Lesson 45: CSS Specificity

Specificity

Specificity determines which CSS rule applies when multiple rules target the same element. Inline styles, IDs, classes, and elements have descending specificity weights. Understanding specificity resolves styling conflicts.

#id {
    color: blue;
}
.class {
    color: red;
}

Blue text (ID wins)

Lesson 46: !important Rule

!important

The !important rule overrides other styles, regardless of specificity, when applied to a property. It should be used sparingly to avoid maintenance issues. !important is a last resort for style conflicts.

.box {
    color: red !important;
}
.box {
    color: blue;
}

Red text (!important wins)

Lesson 47: CSS Inheritance

Inheritance

Some CSS properties, like color or font, are inherited by child elements from their parents. Inheritance simplifies styling for nested elements. Use inherit to explicitly apply a parent’s value.

.parent {
    color: green;
}
.child {
    color: inherit;
}

Green inherited text

Lesson 48: CSS Counters

CSS Counters

CSS counters automatically number elements using counter-increment and counter-reset. They’re displayed via the content property in pseudo-elements. Counters are useful for ordered lists or section numbering.

ol {
    counter-reset: section;
}
li::before {
    counter-increment: section;
    content: counter(section) ". ";
}

Lesson 49: Accessibility with CSS

CSS for Accessibility

CSS can enhance accessibility by ensuring sufficient contrast, focus styles, and readable fonts. Properties like outline and :focus improve keyboard navigation. Accessible CSS ensures all users can interact with your site.

button:focus {
    outline: 2px solid blue;
}
Lesson 50: CSS Clipping

Clipping Paths

The clip-path property shapes an element’s visible area, like circles or polygons. It’s used for creative designs or masking effects. Clipping adds unique visual styles to web elements.

.box {
    clip-path: circle(50% at 50% 50%);
    background-color: teal;
}
